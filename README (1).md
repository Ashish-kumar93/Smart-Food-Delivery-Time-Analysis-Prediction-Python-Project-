# 🍔 Smart Food Delivery Time Analysis & Prediction

## 📌 Project Overview
This project analyzes and predicts food delivery time using real-world factors such as distance, traffic conditions, weather, and city.

The aim is to apply data science techniques including preprocessing, data analysis, visualization, regression modeling, and hypothesis testing to extract meaningful insights.

## 🎯 Objectives
1. Analyze distribution and patterns in delivery time  
2. Examine relationship between distance and delivery time  
3. Evaluate impact of categorical factors (traffic, weather, city)  
4. Analyze correlation among variables  
5. Build and evaluate a Linear Regression model  
6. Perform hypothesis testing (T-test)  
7. Visualize data using graphs  
8. Derive insights and recommendations  

## 📊 Dataset
- projectpython.csv  
- Includes delivery time, traffic, weather, city, and coordinates  

## 🛠️ Technologies Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy  

## 📈 Model
- Linear Regression  
- Metrics: MSE, R² Score  

## 🧪 Hypothesis Testing
- Independent T-test used to compare short vs long distances  

## 📊 Key Insights
- Distance significantly affects delivery time  
- Traffic and weather impact performance  

## 📌 Recommendations
- Optimize routes  
- Avoid peak traffic  
- Improve planning during bad weather
